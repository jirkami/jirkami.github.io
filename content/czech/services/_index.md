## Nabízené služby

Poskytuju komplexní služby v oblasti systémové správy a IT infrastruktury. Každé řešení přizpůsobuji konkrétním potřebám klienta.

### Serverová infrastruktura

#### Linux Server Management
- Instalace, konfigurace a správa Linux serverů (Debian, Ubuntu, CentOS, RHEL)
- Optimalizace výkonu a bezpečnosti
- Aktualizace systémů a aplikací
- Řešení problémů a troubleshooting

#### Windows Server Management
- Active Directory - návrh, implementace, správa
- Group Policy Objects (GPO)
- Windows Server Update Services (WSUS)
- Hyper-V virtualizace
- Exchange Server

#### Virtualizace
- VMware vSphere/ESXi
- Proxmox VE
- Microsoft Hyper-V
- Migrace fyzických serverů do virtuálního prostředí
- Disaster recovery plánování

#### Monitoring a Alerting
- Prometheus + Grafana
- Zabbix
- Nagios/Icinga
- Custom monitoring řešení
- Log management (ELK stack)

### Cloud služby

#### Amazon Web Services (AWS)
- Návrh cloudové architektury
- Migrace do cloudu
- Správa a optimalizace nákladů
- EC2, S3, RDS, Lambda, CloudFormation
- Auto-scaling a high availability
- Security best practices

#### Azure & Exoscale
- Správa Azure zdrojů
- Hybrid cloud řešení
- Migrace workloadů
- Cost management

### Síťová infrastruktura

#### Network Design & Implementation
- Návrh síťové topologie
- VLAN segmentace
- VPN řešení (site-to-site, remote access)
- QoS konfigurace

#### Správa síťového hardware
- Ubiquiti UniFi
- Mikrotik RouterOS
- Cisco switches/routers
- HPE/Aruba

#### VoIP systémy
- Asterisk PBX
- FreePBX
- SIP trunk konfigurace

### Bezpečnost

#### Security Hardening
- CIS Benchmarks implementace
- Firewall konfigurace (iptables, pfSense)
- SSL/TLS certifikáty
- Security audity

#### Compliance
- ISO 27001 podpora
- GDPR compliance
- Bezpečnostní dokumentace
- Risk assessment

#### Backup & Disaster Recovery
- Backup strategie (3-2-1 rule)
- Automatizované zálohování
- Disaster recovery plány
- Testování obnovy dat

### Automatizace a DevOps

#### Infrastructure as Code
- Terraform
- Ansible playbooks
- Packer - automatizace image buildů
- Git workflow

#### CI/CD
- Jenkins
- GitLab CI/CD
- GitHub Actions
- Automated testing

#### Kontejnerizace
- Docker
- Docker Compose
- Kubernetes (základy)
- Container registry

### Podpora uživatelů

- Správa workstations (Windows, macOS, Linux)
- Remote podpora
- Řešení technických problémů
- Školení uživatelů

### Konzultace a poradenství

- Audit současné infrastruktury
- Návrh optimalizací
- Technology selection

- Security best practices
